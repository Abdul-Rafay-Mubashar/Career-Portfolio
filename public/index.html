Tapp Event Solutions

"A Responsive and Scalable Event Management Platform "

Tech Used

React
Rails
Tailwind
Postgress

In spearheading the development of Tapp Event Solutions, my technical contributions were multifaceted and instrumental in meeting the client's objectives. The careful selection of technologies was foundational to the project's success, with React JS and Next JS chosen for the frontend, ensuring a responsive and adaptable user interface. Additionally, the incorporation of Electron JS extended the application's reach across various platforms, accommodating both web and mobile users seamlessly. On the backend, Ruby on Rails was employed for its efficiency and convention-over-configuration principles. This facilitated rapid development, maintaining a clean and scalable codebase. Devise auth was integrated for secure user authentication, laying the groundwork for personalized user experiences and robust permission management. The implementation of a credit-based system, a core feature of Tapp, demanded a nuanced understanding of financial transactions. Integration with popular payment gateways and the use of JSON Web Tokens (JWT) bolstered the development of a secure and efficient credit system, contributing significantly to the platform's functionality. Real-time communication, a crucial aspect for user engagement during events, was achieved through the integration of WebSockets. This facilitated the prompt delivery of notifications, keeping users informed about their orders and pertinent event updates. Testing and quality assurance were prioritized through Test Driven Development (TDD) practices, employing tools like React Testing Library, Capybara, Cypress, and Jest. This meticulous approach ensured the reliability and bug-free performance of every component within the system. To cater to a global audience, internationalization (i18n) and localization (l10n) features were implemented. This strategic move enhanced the platform's accessibility by adapting it to diverse languages and cultural contexts. The integration of OAuth, JSON:API, and RESTful APIs was seamlessly executed, fostering interoperability with third-party services. This not only expanded the platform's functionality but also enriched the user experience by incorporating additional services. Continuous Integration/Continuous Deployment (CI/CD) pipelines were established to automate testing and deployment processes, optimizing development efficiency and ensuring smooth rollouts of updates and new features. In summary, my technical prowess played a pivotal role in the end-to-end development of Tapp Event Solutions, delivering a scalable, secure, and user-friendly platform that met and exceeded the client's expectations in the dynamic landscape of modern event management.

Link Scrape

"LinkedIn Data Scraping Platform"

Vue
FastAPI
Postgress
Bootstrap
Playwright
Gologin Browser

Our LinkedIn Sales Navigator Scraper provides an intelligent and automated solution for extracting valuable business insights from LinkedIn. Designed for efficiency and scalability, the platform processes both person (PURLs) and company (CURLs) links, delivering accurate, structured data directly into the database. The system enables users to create and manage campaigns by uploading URL files along with campaign details such as name, type, and status (Pending, Scheduled). Each campaign can be paused, interrupted, edited (if pending), or deleted, user can also change pending status to inProgress giving users full control over their scraping workflows. Campaign results are securely stored in the database and can be conveniently downloaded in CSV format for further analysis or integration. With its streamlined interface and robust backend, the scraper ensures seamless handling of large-scale LinkedIn searches—such as finding professionals (e.g., Python developers) or exploring target companies. By automating data collection and campaign management, it minimizes manual effort while maximizing accuracy and productivity. This feature-rich, dynamic solution empowers businesses, recruiters, and sales professionals to optimize lead generation, track campaigns effectively, and unlock the full potential of LinkedIn Sales Navigator.

Scraper API
Beautiful Soup

Our German site Car Scraper robust data extraction solution designed to collect detailed automobile information from German car platforms. The system operates in two phases to ensure comprehensive coverage: API-Based Scraping – The scraper connects directly to the platform’s API, retrieving structured vehicle data efficiently and accurately. Website Scraping – Using specific search parameters, the scraper navigates the site to extract additional listings and details not fully exposed through the API. This two-phase approach guarantees high data completeness, combining the speed of API access with the depth of web scraping. The results are processed, structured, and stored for easy analysis or integration into CSV file.
